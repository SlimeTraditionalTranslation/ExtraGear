# 更多裝備
更多裝備是現有的最古老Slimefun附加之一.<br>
功能非常簡單,它為所有Slimefun的錠/合金新增新的武器和護甲.<br>
這些武器與護甲也會自帶附魔!

此為**非官方**版本, 請勿在該作者的問題追蹤內回報!
[原作者連結](https://github.com/TheBusyBiscuit/ExtraGear)
